# Windows
1. 在当前目录下单击左上角【文件】【打开windows powershell】【以管理员身份打开windows powershell】
或者直接以管理员身份运行powershell cd到当前目录

2. 运行以下命令： 

安装

```powershell
powershell -f .\install.ps1
```

卸载

```powershell
powershell -f .\uninstall.ps1
```



# Linux
在当前目录运行以下命令：

安装

```shell
sudo sh .\install.sh
```

卸载

```shell
sudo sh .\uninstall.sh
```
